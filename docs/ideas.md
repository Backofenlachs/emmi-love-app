# Emmi App – Ideen Sammlung

## Emotion / UX

- First Open Effekt (einmalige besondere Nachricht)
- Letzte Nachricht = „Special Screen“ (anderes Layout, ruhiger Moment)
- Nachricht langsam „eintippen“ (Typing Animation)
- Hintergrund ändert sich je nach Nachricht (z. B. Sonnenuntergang, Nacht)
- Kleine „Überraschung“ nach X Klicks

---

## Interaktionen

- Herz klick → kleine Partikel / Herz-Explosion
- Swipe nach links/rechts → nächste Nachricht
- Long Press auf Herz → spezielle Nachricht
- Doppel-Klick → „Ich liebe dich“ Overlay

---

## Visual Effects

- Floating Hearts im Hintergrund
- Glowing Herz (leichtes Leuchten)
- Soft Blur / Depth Effekt
- Gradient Animation (Background bewegt sich leicht)

---

## PWA / App Features

- Splash Screen optimieren (schöner Einstieg)
- App Icon verbessern (custom statt Emoji)
- Version anzeigen (für Updates sichtbar)
- „Neue Version verfügbar“ Hinweis

---

## Persönlicher Touch

- Eigene Nachrichten hinzufügen (config file)
- Name dynamisch (nicht hardcoded)
- Datum einbauen („Seit X Tagen zusammen“)
- Eigene Bilder einbauen (z. B. Galerie)

---

## Special Features

- „Memory Mode“ → zeigt alte Nachrichten zufällig
- „Random Love Message“ Button
- Mini-Tagebuch (1 Satz pro Tag speichern)
- Hidden Message (Easter Egg)

---

## Tech / Architektur

- Nachrichten aus JSON laden (sauber trennen)
- kleine State-Struktur (z. B. currentState: intro / normal / special)
- Animationen modularisieren
- Config für Theme (Farben austauschbar)

---

## Spätere Ideen (optional größer gedacht)

- Verbindung mit Backend (z. B. Nachrichten live ändern)
- Push Notifications (z. B. „Denk an dich“)
- Multi-Page (z. B. „Unsere Momente“)

---

## Notizen

- Fokus bleibt: simpel + emotional
- nicht überladen → Qualität > Features
- jede Änderung soll „spürbar“ sein, nicht nur technisch
